About This Portal

Single-file standalone portal. Upload to SharePoint or any web server — no iframes, no dependencies, no server needed.

How to Use

Step 1Click Launch Module 1 to track UAT test case progress. Upload Excel or CSV.
Step 2Columns needed: Test Case ID, Name, Tester, Status, Start Date, End Date.
Step 3Click Launch Module 2 for issue & SLA monitoring. Upload issue tracker export.
Step 4Use filters, date ranges, status chips. Export PDF reports as needed.
